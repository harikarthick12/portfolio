# Hari Portfolio (Futuristic)
This is a ready-to-run React + Vite + Tailwind portfolio created for Hari Karthick.
Features: custom cursor, framer-motion animations, glassmorphism, responsive layout.

## How to run locally
1. `npm install`
2. `npm run dev`

## Deploy
Connect this repo to Netlify; the included `netlify.toml` uses `npm run build` and publishes `dist`."# portfolio"
